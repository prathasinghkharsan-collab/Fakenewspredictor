# Fake News Detection AI App

## Overview
This project is a **Fake News Detection AI** built using Python and Streamlit.  
It predicts whether a news article is **REAL** or **FAKE**, shows **confidence scores**, and highlights the **most influential words**.  
Interactive graphs are included for better visualization.

---

## Features
- Input news text manually or upload a small CSV (demo-friendly).  
- Predicts news **label** (REAL or FAKE).  
- Shows **prediction confidence** in percentage.  
- Highlights **top contributing words** from TF-IDF.  
- Interactive graphs for exploring predictions.  
- Polished Streamlit interface for presentations.
